# Learning Logs

This directory contains daily and weekly learning logs to track your progress.

## Format

You can organize your logs by date or topic. Here's a suggested format:

### Daily Log Template

```markdown
# Date: YYYY-MM-DD

## What I Learned Today
- Topic 1
- Topic 2

## Challenges Faced
- Challenge and how you overcame it

## Resources Used
- Links to tutorials, documentation, etc.

## Tomorrow's Goals
- What you plan to learn next
```

### Weekly Log Template

```markdown
# Week of YYYY-MM-DD to YYYY-MM-DD

## Summary
Brief overview of the week's learning

## Key Achievements
- Achievement 1
- Achievement 2

## Time Spent
- Topic A: X hours
- Topic B: Y hours

## Next Week's Focus
- Goals for next week
```

## Example

Create files like:
- `2026-01-30.md` for daily logs
- `week-2026-W05.md` for weekly logs
