# Learning Projects

This directory contains hands-on projects and exercises to apply what you're learning.

## Project Organization

Each project should have its own folder with:
- Source code
- README with project description
- Documentation
- Any relevant resources

## Project Template Structure

```
project-name/
├── README.md          # Project overview and instructions
├── src/              # Source code
├── docs/             # Additional documentation
├── tests/            # Test files (if applicable)
└── resources/        # Images, data files, etc.
```

## Project README Template

```markdown
# Project Name

## Description
Brief description of what this project does

## Learning Goals
- Goal 1
- Goal 2

## Technologies Used
- Technology 1
- Technology 2

## Setup Instructions
1. Step 1
2. Step 2

## Usage
How to run/use the project

## Challenges & Solutions
- Challenge faced and how you solved it

## Next Steps
- Potential improvements or extensions

## Resources
- Links to tutorials or references used
```

- Small coding challenges
- Tutorial follow-alongs
- Original projects applying concepts
